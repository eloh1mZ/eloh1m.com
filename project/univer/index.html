Univer

-приложение для просмотра оценок, расписания занятий и экзаменов, и другой информации о студенте.

  • — Прогрессивное веб-приложение
  • Svelte
  • AIOHTTP
  • BeautifulSoup

Клиентская часть написана на Svelte, используя собственно созданную библиотеку компонентов. Серверная часть на Python. В качестве HTTP-сервера используется AIOHTTP. Данные получаются с сайта информационной системы Univer 2.0, производится парсинг и результат отдаётся пользователю. Для парсинга используется библиотека BeautifulSoup.
